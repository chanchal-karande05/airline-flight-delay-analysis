# Dataset Source

The flight dataset used in this project is large and exceeds GitHub file size limits.

You can download the dataset from the following source:

https://www.kaggle.com/datasets/usdot/flight-delays

Files used in this project:

- flights.csv
- airlines.csv
- airports.csv

After downloading the dataset, import the CSV files into Microsoft SQL Server
and run the SQL queries provided in the `sql_queries` folder.
